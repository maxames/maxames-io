# maxames.io

Personal website monorepo with two apps:

- **apps/main** - Main landing page (maxames.io)
- **apps/notes** - Blog (notes.maxames.io)

## Development

```bash
# Run main site
pnpm dev

# Run notes blog
pnpm dev:notes

# Build
pnpm build
pnpm build:notes
```
